Horizontal tabs plugin enable to show unlimited posts in 3 tabs

== Description ==

With horizontal tabs plugin you can add 3 tabs that each of them contains unlimited posts. Each tab shows the featured image and the title of
each post


* option to select the category of post in each tab
* option to select the width and the height of feautured image
* option to select the number of post in tabs
 

== Installation ==


1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
